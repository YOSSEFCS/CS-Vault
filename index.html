يجب إعداد Firebase أولاً

افتح ملف index.html وابحث عن التعليق FIREBASE CONFIG واستبدل البيانات ببيانات مشروعك من لوحة Firebase:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "project-id.firebaseapp.com", projectId: "project-id", storageBucket: "project-id.appspot.com", messagingSenderId: "1234567890", appId: "1:123...:web:abc..." };

بعد التعديل احذف هذا التنبيه من الكود (div id="configNotice")
